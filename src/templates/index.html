Добро пожаловать на Awesome File Sharing!!

Это простенький файлообменник на JavaScript и PHP с открытым исходным кодом для обмена файлами как не странно :>

Чем можно обмениваться?

  • Изображения (*.png, *.jpeg, *.jpg, *.gif, ...)
  • Видео (*.mp4, *.webm, *.3gp, ...)
  • Аудио (*.mp3, *.ogg, *.m4a, ...)
  • Документы (*.txt, *.py, *.js, *.html, ...)
  • Архивы (*.zip)
